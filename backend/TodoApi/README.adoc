== Experimenting with the repl and swagger

* re-enabled Swagger in `Program.cs`

* connect using just:
+
[source,powershell]
----
httprepl https://localhost:7020/
----
+
[source,output]
----
❯ httprepl https://localhost:7020/
(Disconnected)> connect https://localhost:7020/
